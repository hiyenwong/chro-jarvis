## 商店列表信息

### 简短描述（最多 132 字符）

```
AI 驱动的智能 Chrome 扩展，提供问答、翻译和 Token 监控功能
```

### 详细描述

```
Chro-Jarvis 是一个功能强大的 AI Chrome 扩展，集成了智能问答、页面翻译和 Token 消耗监控功能。

🤖 AI 问答功能
• 基于页面上下文的智能问答
• 支持 DeepSeek、智谱 AI、火山引擎、OpenAI、Claude
• 每个标签页独立的对话历史

🌍 翻译功能
• 一键翻译整页
• 支持中文、英语、日语、韩语、法语、德语、西班牙语
• 大型页面分批处理，显示实时进度

📊 Token 监控
• 实时 Token 使用统计
• 按时间段、提供商、模型的多维分析
• 可视化趋势图表
• 数据导出/导入功能

✨ 用户体验
• Toast 通知系统
• 优雅的错误处理
• 响应式界面设计

安装后只需配置 API Key 即可开始使用。所有数据仅存储在本地，保护您的隐私。

支持 Chrome 88 及以上版本。
```

### 屏幕截图说明

需要准备以下尺寸的截图：

1. **1280x800** 或 **640x400** - 主功能展示（必需）
   - 展示侧边栏 AI 问答界面
   - 展示页面翻译效果

2. **1280x800** 或 **640x400** - Token 统计面板（必需）
   - 展示统计数据和趋势图

3. **1280x800** 或 **640x400** - 设置界面（必需）
   - 展示 AI 提供商选择
   - 展示 API Key 配置

4. **1280x800** 或 **640x400** - 快速操作（可选）
   - 展示弹出窗口的快速操作按钮

### 图标要求

需要以下尺寸的图标：

- **16x16** - 浏览器工具栏图标
- **48x48** - 扩展管理页面图标
- **128x128** - Chrome Web Store 图标
- **可选**: 256x256、512x512 高分辨率图标

**图标设计建议**：
- 使用简洁的字母 "J" 或机器人头像
- 主色调：蓝色 (#4f46e5) 或绿色 (#10b981)
- 确保在小尺寸下清晰可辨

## 隐私政策

由于扩展将数据存储在本地，需要创建简单的隐私政策：

### 隐私政策摘要

```
Chro-Jarvis 隐私政策

Chro-Jarvis 非常重视用户隐私。本扩展：

1. 数据存储
   - 所有数据仅存储在您的浏览器本地
   - API Key 使用 Chrome 加密存储
   - 不会上传到任何服务器

2. 网络通信
   - 仅与您选择的 AI 提供商通信
   - 使用 HTTPS 加密传输
   - 不经过任何中间服务器

3. 数据收集
   - 不收集任何个人数据
   - 不追踪用户行为
   - 不使用分析工具

4. 第三方服务
   - 直接与 AI 提供商 API 通信
   - 受各提供商的隐私政策约束
   - 建议查看提供商的隐私政策

5. 用户权利
   - 您可以随时清除本地数据
   - 您可以随时卸载扩展
   - 卸载后所有本地数据将被删除

联系我们：support@chro-jarvis.com
最后更新：2026-03-01
```

## 发布步骤

1. **准备开发者账号**
   - 注册 Chrome Web Store 开发者账号
   - 支付 $5 一次性费用

2. **创建新项目**
   - 访问 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
   - 点击「新建项目」

3. **上传扩展包**
   - 将 `dist` 目录打包为 ZIP
   - 上传 ZIP 文件

4. **填写商店列表信息**
   - 上传图标和截图
   - 填写描述和分类
   - 添加标签

5. **提交审核**
   - 检查所有信息
   - 提交审核
   - 等待审核结果（通常 1-3 天）
