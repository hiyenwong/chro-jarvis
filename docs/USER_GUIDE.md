## 快速入门

### 第一步：安装扩展

1. 下载扩展文件或从源码构建
2. 在 Chrome 中打开 `chrome://extensions/`
3. 启用「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `dist` 文件夹

### 第二步：配置 API Key

1. 点击浏览器工具栏中的 Chro-Jarvis 图标
2. 切换到「设置」标签
3. 选择 AI 供应商（推荐使用 DeepSeek）
4. 输入您的 API Key
5. 点击「保存配置」

> **💡 提示**：首次使用需要先获取 AI 提供商的 API Key。详见 [README.md](../README.md#api-key-获取指南)

### 第三步：开始使用

现在您可以：
- 打开侧边栏进行 AI 问答
- 翻译整个网页
- 查看 Token 使用统计

## AI 问答功能

### 基础使用

1. **打开侧边栏**
   - 点击扩展图标
   - 选择「打开侧边栏」

2. **提问**
   - 在输入框中输入您的问题
   - 点击发送或按 Enter 键
   - AI 将基于当前页面内容回答

### 高级技巧

| 技巧 | 说明 |
|------|------|
| 📄 页面摘要 | 输入「请总结这个页面的主要内容」 |
| 🔍 内容提取 | 输入「提取这篇文章中的关键信息」 |
| 📝 内容改写 | 输入「将这段话改写成更简洁的形式」 |
| 🌐 语言转换 | 输入「将这段英文翻译成中文」 |

### 对话历史

- 每个标签页的对话历史是独立的
- 切换标签页时，会自动加载该页面的对话历史
- 对话历史保存在本地，不会上传

## 翻译功能

### 翻译整页

1. **点击扩展图标**
2. **选择「翻译页面」按钮**
3. **等待翻译完成**

### 翻译设置

在「设置」标签中可以：
- 更改默认目标语言
- 选择翻译质量偏好

### 翻译进度

- 翻译大页面时会显示进度
- 可以随时取消翻译
- 翻译完成后会显示 Toast 通知

### 支持的语言

- 中文（简体、繁体）
- 英语
- 日语
- 韩语
- 法语
- 德语
- 西班牙语

## Token 统计

### 查看统计

1. 点击扩展图标
2. 切换到「Token 统计」标签

### 统计维度

| 维度 | 说明 |
|------|------|
| ⏱️ 时间范围 | 今天 / 7天 / 30天 / 全部 |
| 🤖 提供商 | DeepSeek、智谱、火山等 |
| 🎯 模型 | 各提供商的具体模型 |
| 📈 趋势 | 每日使用趋势图表 |

### 成本计算

- 自动计算 Token 成本
- 基于各提供商的最新定价
- 支持自定义价格设置

### 数据管理

- **导出数据**：导出 JSON 格式的统计记录
- **导入数据**：导入之前导出的记录
- **清空数据**：清除所有统计数据

## 常见问题

### ❓ 扩展无法加载？

**解决方案**：
1. 检查 Chrome 版本是否 ≥ 88
2. 重新加载扩展：`chrome://extensions/` → 点击「重新加载」
3. 查看控制台错误信息

### ❓ API 调用失败？

**可能原因**：
- API Key 无效或过期
- 网络连接问题
- API 额度已用完

**解决方案**：
1. 检查 API Key 是否正确
2. 测试网络连接
3. 查看账户余额和额度

### ❓ 翻译质量不好？

**改善方法**：
1. 切换到更强大的模型（如 Claude）
2. 对于专业术语，可以提示 AI 注意术语翻译
3. 分段翻译长文本

### ❓ Token 统计不准确？

**说明**：
- Token 统计基于 AI 提供商返回的数据
- 不同提供商的 Token 计算方式可能不同
- 统计仅供参考，实际计费以提供商账单为准

## 技巧与窍门

### 💡 效率提升

1. **快捷键**（即将推出）
   - `Alt + J`: 打开侧边栏
   - `Alt + T`: 翻译页面

2. **批量操作**
   - 在多个标签页中同时使用
   - 每个标签页独立记录对话

3. **成本控制**
   - 定期查看 Token 统计
   - 选择性价比高的提供商
   - 删除不需要的对话历史

### 🎯 最佳实践

1. **提问技巧**
   - 问题越具体，答案越准确
   - 可以引用页面内容进行追问
   - 利用对话历史进行上下文连贯的交流

2. **翻译建议**
   - 技术文档建议使用专业模型
   - 日常内容可以使用性价比高的模型
   - 翻译后可以手动调整

3. **Token 节省**
   - 避免重复翻译相同内容
   - 合理使用对话历史
   - 定期清理统计数据
